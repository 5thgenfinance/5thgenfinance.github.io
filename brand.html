Print Margins

Outer page margins: 1 inch (72pt) on all sides for print documents.

Column Grid

Use a 12-column grid for digital layouts. Primary content 8-col, sidebar 4-col.

Dividers

Use a 1px bronze (#c49a4a) horizontal rule between major sections.

Background Surfaces

Never use color backgrounds other than Navy #1a2744 or Off-White #f7f4ef for primary layout surfaces.

Usage Guidelines

Consistent application of these rules protects the integrity and recognizability of the Fifth Generation Advisers brand.

Navy for Primary Headers

Use navy for all primary headers and formal document backgrounds.

Bronze as Accent Only

Use bronze only as an accent — rules, borders, icon highlights.

Consistent Grid Spacing

Maintain consistent 8px grid spacing across all materials.

No Busy Backgrounds

Do not use the logo on busy photographic backgrounds without a clear zone.

No Typeface Substitutions

Do not substitute typefaces — Playfair Display and Source Sans Pro are required.

No Effects on Logo

Do not use gradients, drop shadows, or decorative effects on the logo.
